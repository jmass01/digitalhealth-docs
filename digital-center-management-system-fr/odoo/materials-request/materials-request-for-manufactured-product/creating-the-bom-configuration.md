# Création de la configuration de nomenclature (BoM)

Pour configurer une **nomenclature (BoM)**, commencez par sélectionner un **Utilisateur de Service (SU)** :

1. Allez dans le menu **Gestion des utilisateurs de service**.
2. Repérez le SU souhaité dans la liste et cliquez dessus pour ouvrir son formulaire.

### Étapes de création de la BoM

1. Dans le formulaire, cliquez sur le bouton **Configuration MRP**.
   * Ce bouton affiche le nombre de configurations MRP déjà créées pour cet US.
   * En cliquant dessus, vous verrez la liste des configurations existantes.
2. Cliquez sur **Créer** (en haut à gauche) pour initier une nouvelle configuration.
3. **Sélectionnez le modèle de BoM** approprié pour la configuration :
   * Seuls les modèles approuvés apparaissent dans le menu déroulant.
   * Exemple : pour une prothèse trans-tibiale (PTB).
   * Pour créer ou approuver un modèle de BoM, reportez-vous à la section **Modèle MRP**.
4. **Sélectionnez les attributs du produit** :
   * Chaque modèle propose différents attributs à renseigner (ex. côté, taille, couleur).
   * Lors de la saisie du côté et de la taille, la variante de produit se met automatiquement à jour.
5. Cliquez sur **Confirmer la configuration**, puis sur **Créer la BoM**.
6. En cas de message contextuel, il peut indiquer l’un des problèmes suivants :
   * Un produit n’est pas correctement configuré.
   * Un composant est manquant ou archivé.
   * Un produit n’a pas un attribut spécifique (souvent couleur ou taille).

> Pour résoudre ces problèmes, consultez la section [Dépannage](../../../depannage-et-faq/)
